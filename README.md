# Networking-Troubleshooting-Lab
Hands-on networking troubleshooting lab demonstrating real-world issue diagnosis using commands like ipconfig, ping, tracert, and nslookup


## Overview
This project demonstrates hands-on network troubleshooting using common command-line tools. It simulates real-world IT support scenarios such as DNS issues, connectivity failures, and slow network performance.

## Tools Used
- Command Prompt
- ipconfig
- ping
- tracert
- nslookup

## Skills Demonstrated
- Network diagnostics
- DNS troubleshooting
- IP configuration analysis
- Connectivity issue resolution

## Outcome
Developed practical troubleshooting skills required for IT Support and Help Desk roles.
